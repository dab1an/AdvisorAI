# Frontend

### Running
Navigate to `client` folder

```
npm i
npm run dev
```

# Backend 

Install a .NET compatible IDE like `Rider` or `Visual Studio` (not VSC)

Navigate to `server` folder

Add OpenAI key in `appsettings.json`

In terminal run `dotnet run`
